Para la construcción de este proyecto se utilizaron las siguientes tecnologías:

1. IDE: IntelliJ IDEA
2. Lenguaje: Java
3. Framework: JavaFX
4. Backend: NestJS
5. Base de datos: PostgreSQL
6. ORM: TypeORM
7. Control de versiones: Git
